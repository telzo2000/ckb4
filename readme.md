# ckb4

![](img/img00001.jpg)

## What is ckb4?

ckb4 is a small keyboard with qaz layout.
<br>
ckb4 has one joystick and one rotally encoder.
<br>
ckb4 use RP2040-Zero,run qmk_firmware.
<br>
<br>

## firmware

[vial](https://get.vial.today/) is possible.
<br>
